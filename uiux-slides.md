# Slides Skill

Erstellt datengetriebene HTML-Präsentationen mit professionellen Layouts und Copywriting-Frameworks.

## Verwendung

```
/uiux-slides [Thema] [Anzahl-Slides]
```

Beispiel: `/uiux-slides Produkt-Pitch Q2 12`

## Use Cases

- Marketing-Präsentationen und Pitch Decks
- Daten-Visualisierungen mit Chart.js
- Strategische Slide-Designs mit etablierten Layout-Patterns
- Persuasiver Content mit Copywriting-Best-Practices

## Referenz-Materialien

- **Layout Patterns** — Strukturelle Design-Ansätze
- **HTML Template** — Grundlegendes Markup
- **Copywriting Formeln** — PAS, AIDA, FAB-Frameworks
- **Slide Strategies** — Kontextuelle Präsentations-Ansätze

## Workflow

1. Befehl parsen (Thema + Slide-Anzahl)
2. Relevante Referenz-Materialien laden
3. Präsentation mit Design Tokens und Chart.js generieren
4. Navigations-Controls einbauen
