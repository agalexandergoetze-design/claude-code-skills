# UI Styling Skill

Baut accessible, professionelle UIs mit shadcn/ui, Tailwind CSS und React.

## Core Stack

- **shadcn/ui** — Copy-paste Komponenten auf Radix UI Primitives (TypeScript)
- **Tailwind CSS** — Utility-first Styling
- **Canvas** — Visual Design für komplexe Grafiken

## Setup

```bash
npx shadcn@latest init
```

Dann Komponenten selektiv hinzufügen:
```bash
npx shadcn@latest add button card dialog
```

## Wann verwenden

- React-Anwendungen mit accessible Komponenten
- Forms, Tables, Dialogs, Navigations
- Responsive Layouts (mobile-first)
- Dark Mode Support
- Design System Implementierung

## Design-Philosophie

"Museum-quality visual compositions" — sophistizierte visuelle Kommunikation mit minimalem Text. UI-Entwicklung als Handwerk.

## Best Practices

- Komponenten-Komposition über Custom-Code
- Accessibility-first via Radix Primitives
- Mobile-first Responsive Design
- Konsistente Design Tokens
- Dark/Light Mode separat testen
