# Meine Claude Skills

| Command | Was es macht |
|---------|-------------|
| `/erklaer` | Erklärt markierten Code auf Deutsch |
| `/fix` | Analysiert und behebt einen Bug |
| `/tests` | Schreibt Tests für markierten Code |
| `/refactor` | Verbessert Struktur ohne Verhaltensänderung |
| `/commit` | Erstellt eine Git-Commit-Message |

## Superpowers Skills (von github.com/obra/superpowers)

| Command | Was es macht |
|---------|-------------|
| `/superpowers-brainstorm` | Strukturierter Brainstorming-Prozess → Design → Plan (kein Code ohne Approval) |
| `/superpowers-plan` | Erstellt detaillierten Implementierungsplan mit TDD-Schritten |
| `/superpowers-execute` | Führt einen geschriebenen Plan Schritt für Schritt aus |
| `/superpowers-tdd` | Test-Driven Development — erst Test, dann Code |
| `/superpowers-debugging` | Systematisches Debugging — Root Cause first, kein Raten |
| `/superpowers-verify` | Verifiziert Arbeit mit echten Befehlen vor jeder Fertig-Meldung |

## Eigene Skills hinzufügen

Neue `.md`-Datei in `~/.claude/commands/` erstellen → sofort als `/dateiname` verfügbar.
