# Design Skill

Umfassendes Design-Toolkit. Delegiert spezialisierte Aufgaben an Sub-Skills.

## Capabilities

- **Logo Design** — 55+ Stile, 30 Farbpaletten, 25 Branchen-Guides (via Gemini AI)
- **Corporate Identity (CIP)** — 50+ Deliverables, 20 Stile, 20 Branchen, Mockups + HTML-Präsentation
- **Präsentationen** — Strategische HTML-Slides mit Chart.js und Design-Token-Integration
- **Banner** — 22 Art-Direction-Stile für Social, Ads, Web und Print
- **Icons** — SVG-Icons in 15 Stilen (Gemini 3.1 Pro), Text-Output
- **Social Photos** — Multi-Plattform-Workflow: HTML/CSS → Screenshot in Plattform-Dimensionen

## Routing

- Brand Identity → `/uiux-brand`
- Design Tokens → `/uiux-design-system`
- UI-Code → `/uiux-styling`
- Logo, CIP, Slides, Banner, Icons, Social → direkt hier

## Workflow

1. Aufgabe analysieren und an richtigen Sub-Skill delegieren
2. Bei Logo: Stil + Branche + Farbpalette erfragen
3. Bei CIP: Vollständiges Paket Logo → Mockups → Pitch-Deck
4. Bei Social: Analyse → Ideation → Design → Export → Verifikation
